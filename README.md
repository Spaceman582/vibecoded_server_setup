# vibecoded_server_setup
Быстрое развёртывание сервера, с автоматической настройкой входа, файервола и тому подобного

##ВАЖНО:
Перед использованием скрипта, добавьте публичный ключ на сервер по пути /root/.ssh/authorized_keys, либо добавьте его автоматически с помощью Termius

Запуск
```
bash <(curl -Ls https://raw.githubusercontent.com/Spaceman582/vibecoded_server_setup/refs/heads/main/setup.sh)
```
Тестировалось только на Ubuntu
